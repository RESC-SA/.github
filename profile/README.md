## About Us

**RESC** (Renewable Energy & Smart Control) is a Saudi Arabian technology company specializing in energy systems, power electronics, and industrial automation. We design hardware, firmware, and software solutions that power smart energy infrastructure across the Kingdom.

---

## What We Do

| Domain | Description |
|---|---|
| ⚡ **Energy Systems** | Solar inverters, combiner boxes, and energy management units |
| 🔌 **Power Electronics** | Firmware for motor controllers, inverters, and smart switchgear |
| 📱 **Mobile Applications** | Flutter apps for real-time monitoring and field diagnostics |
| 🛠️ **Embedded Firmware** | C/C++ firmware for microcontrollers via Modbus, CAN, and BLE |
| 🌐 **Industrial IoT** | Cloud-connected gateways for remote asset monitoring |
| 🏗️ **Industrial Automation** | Smart control panels and SCADA-ready devices |

## Our Products

### CBX Series — Smart Combiner Box Controllers
Intelligent solar PV combiner box solutions with string monitoring, DC protection, and Modbus communication for Saudi and GCC solar farms.

- **CBX-Controller-FW** — Main controller embedded firmware
- **CBX-Interface-Board-FW** — HMI and RS485/Modbus communication firmware
- **Combiner Module (C)** — C library for power measurement and fault detection

### Inverter Gateway — Flutter Mobile App
Bluetooth-enabled mobile app for solar inverter commissioning and diagnostics using Modbus RTU over BLE.
