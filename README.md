## About Us

RESC is a Saudi Arabian technology company specializing in **energy systems**, **power electronics**, and **industrial automation**. We design and develop hardware, firmware, and software solutions that power the next generation of smart energy infrastructure across the Kingdom.

Our interdisciplinary team combines deep expertise in embedded systems, renewable energy engineering, and mobile/cloud application development to deliver end-to-end solutions — from circuit board to user interface.

## Our Products

### CBX Series — Combiner Box Controllers
Intelligent combiner box solutions for solar PV systems. The CBX line integrates string monitoring, protection, and remote diagnostics into a compact, field-ready enclosure.

- **CBX-Controller-FW** — Embedded firmware for the main controller board
- **CBX-Interface-Board-FW** — Firmware for the HMI and communication interface module
- **Combiner Module (C)** — Core C library for power measurement and protection logic

### Inverter Gateway
A Bluetooth-enabled mobile gateway application allowing field engineers to configure, monitor, and diagnose solar inverters on-site — built on Flutter with Modbus RTU over BLE.
